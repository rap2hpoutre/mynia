# mynia
Unicorn name generator. Give a name to your favorite unicorn.

## Install
```
cargo build --release
````
## Usage
```
raph:~ raphael$ mymia
Your unicorn name will be: Yojo

raph:~ raphael$ mymia
Your unicorn name will be: Leendo

raph:~ raphael$ mymia
Your unicorn name will be: Vipe

raph:~ raphael$ mymia
Your unicorn name will be: Mimba
```
